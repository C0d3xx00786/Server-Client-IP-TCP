Здесь находится две программы: первая — это проверка подключения к сайту, а вторая — это получение данных как Клиент из Сервера, что помогает проверить работоспособность и состояние сервера.

Стоит уточнить, что библиотека, которая используется для получения информации о скорости сети, работала через Speedtest, но, как вы, наверно, уже знаете, Speedtest у нас не работает, так что при включении программы Server и Client лучше включить VPN.
